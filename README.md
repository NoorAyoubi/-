🚀 JLM Legal Bridge
An AI-powered digital intake platform designed to simplify and improve access to legal assistance for injury cases, especially for Arabic-speaking users in East Jerusalem.

💡 Overview
JLM Legal Bridge provides a smart conversational experience that helps users quickly understand whether they may have a legal case. Instead of complex forms or immediate lawyer contact, users interact with an intelligent chatbot that guides them step-by-step.
The system reduces fear, confusion, and friction, making it easier for users to take their first step toward legal support.

⚙️ Key Features


🤖 AI Chatbot Interaction
Dynamically adapts questions based on user responses for a smooth and personalized experience.


⚡ Instant Case Evaluation
Provides a quick preliminary assessment so users understand their situation within minutes.


🎙️ Voice-to-Text Input
Allows users to describe their case verbally, improving accessibility and usability.


🔐 Privacy & Consent Control
Users decide whether to send their information or delete it completely.


📊 Lawyer Dashboard
Displays structured case summaries for efficient review and decision-making.



🧠 Problem Solved
Many injured individuals avoid contacting lawyers due to:

Financial anxiety
Language barriers
Legal complexity
Lack of trust

This platform turns the process into a simple, safe, and guided experience.

🛠️ Tech Stack

Frontend: HTML, CSS, JavaScript
AI Integration: Chat-based logic (LLM-ready)
Backend / Storage: Firebase (or basic storage for MVP)
Dashboard: Custom web dashboard for lawyers
